# Übung SQL

1. Geben Sie die Namen aller Kuenstler aus

```sql
SELECT name from artists;
```

2. Geben Sie die Anzahl aller Kuenstler aus

```sql
SELECT count(name) as "Anzahl Kuenstler" from artists;
```

3. Geben Sie die Künstlernamen aus, der an beliebiger Stelle das Teilwort guns beinhaltet

```sql
SELECT name from artists where name like '%guns%';
```

4. Geben Sie alle Alben und ihre Künstler aus, die im Albumtitel irgendwo das Teilwort rock beinhaltet

```sql
SELECT albums.Title, artists.name from artists INNER JOIN albums on artists.ArtistId = albums.Artistid where albums.Title like '%rock%';
```

5. Geben Sie die Anzahl der Alben von Iron Maiden an, die im Albumtitel irgendwo das Teilwort death beinhalte


```sql
SELECT count(albums.Title) as "Anzahl der Alben" from artists INNER JOIN albums on artists.ArtistId = albums.Artistid where albums.Title like '%death%' and artists.name = "Iron Maiden";
```

7. Zu wie vielen unterschiedlichen Staedten wurde Rechnungen in Deutschland geschickt?


12. Welche InvoiceId s gibt es, deren Tracks auf Alben mit der KuenstlerId 3 verweisen und deren Track min. 70000
ms lang ist?

```sql
SELECT DISTINCT InvoiceId FROM invoice_items INNER JOIN tracks on invoice_items.TrackId = tracks.TrackId INNER JOIN albums on tracks.AlbumId = albums.AlbumId WHERE albums.ArtistId = 3 AND tracks.Milliseconds >= 70000;
```

13. Wie viele Musikstücke gibt es, die keinen Komponisten angegeben haben und die dem Genre des Jazz
zuzuordnen sind?

```sql
SELECT count(tracks.Name) as 'Anzahl Stücke' FROM tracks INNER JOIN genres on tracks.GenreId = genres.GenreId where tracks.Composer is NULL AND genres.Name = 'Jazz';
```

14. Wie viele Musikstücke gibt es, die dem Genre des Drama zuzuordnen sind und die nicht die AlbumId 228
besitzen?

```sql
SELECT count(tracks.Name) as 'Anzahl Stücke' FROM tracks INNER JOIN genres on tracks.GenreId = genres.GenreId AND genres.Name = 'Drama' AND tracks.AlbumID <> 228;
```

15. Welche Künstler haben mehr als drei Alben?
 
```sql
SELECT artists.Name from artists INNER JOIN albums on artists.ArtistId = albums.ArtistId GROUP BY artists.ArtistId HAVING count(artists.Name) > 3;
```
