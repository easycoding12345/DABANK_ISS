01.  Geben Sie die Namen aller Kuenstler aus

```SQL

```

02.  Geben Sie die Anzahl aller Kuenstler aus

03.  Geben Sie die Künstlernamen aus, der an beliebiger Stelle das Teilwort guns beinhaltet

```SQL

```

04.  Geben Sie alle Alben und ihre Künstler aus, die im Albumtitel irgendwo das Teilwort rock beinhaltet

```SQL
SELECT albums.Title, artists.name from artists INNER JOIN albums on artists.ArtistId = albums.Artistid where albums.Title like '%rock%';
```

05.  Geben Sie die Anzahl der Alben von Iron Maiden an, die im Albumtitel irgendwo das Teilwort death beinhalte

```SQL
SELECT count(albums.Title) as "Anzahl der Alben" from artists INNER JOIN albums on artists.ArtistId = albums.Artistid where albums.Title like '%death%' and artists.name = "Iron Maiden";
```

12.  Welche InvoiceId s gibt es, deren Tracks auf Alben mit der KuenstlerId 3 verweisen und deren Track min. 70000  
    ms lang ist?

```SQL
SELECT DISTINCT InvoiceId FROM invoice_items INNER JOIN tracks on invoice_items.TrackId = tracks.TrackId INNER JOIN albums on tracks.AlbumId = albums.AlbumId WHERE albums.ArtistId = 3 AND tracks.Milliseconds >= 70000;
```

13.  Wie viele Musikstücke gibt es, die keinen Komponisten angegeben haben und die dem Genre des Jazz  
    zuzuordnen sind?

```SQL
SELECT count(tracks.Name) as 'Anzahl Stücke' FROM tracks INNER JOIN genres on tracks.GenreId = genres.GenreId where tracks.Composer is NULL AND genres.Name = 'Jazz';
```

14.  Wie viele Musikstücke gibt es, die dem Genre des Drama zuzuordnen sind und die nicht die AlbumId 228  
    besitzen?

```SQL
SELECT count(tracks.Name) as 'Anzahl Stücke' FROM tracks INNER JOIN genres on tracks.GenreId = genres.GenreId AND genres.Name = 'Drama' AND tracks.AlbumID <> 228;
```

15.  Welche Künstler haben mehr als drei Alben?

```SQL
SELECT artists.Name from artists INNER JOIN albums on artists.ArtistId = albums.ArtistId GROUP BY artists.ArtistId HAVING count(artists.Name) > 3;
```

16.  Geben Sie alle Playlists und die Anzahl derer Tracks, sortiert nach der Anzahl der Tracks pro Playlist aus

```SQL
SELECT name, count(name) as "Anzahl Tracks" FROM playlists INNER JOIN playlist_track on playlists.PlaylistId = playlist_track.PlaylistId GROUP BY playlists.Name ORDER BY count(playlists.Name) DESC
```

18. Das Minimum, Maximum, Mittelwert und Gesamtanzahl aller Rechnungen die nach Norwegen gestellt wurden

19. Die Anzahl an Tracks die in der Playlist Heavy Metal Classic enthalten sind und im AAC Format gespeichert 
wurden
```SQL
SELECT COUNT(tracks.TrackId) AS "Anzahl Tracks im AAC Format"
FROM tracks
INNER JOIN playlist_track ON tracks.TrackId = playlist_track.TrackId
INNER JOIN playlists ON playlist_track.PlaylistId = playlists.PlaylistId
INNER JOIN media_types ON tracks.MediaTypeId = media_types.MediaTypeId
WHERE playlists.Name = 'Heavy Metal Classic' AND media_types.Name = 'AAC audio file';
```

20. Alle Länder in denen Rechnungen gestellt wurden, deren Summe aller Rechnungen mehr als 45 Dollar ist
```SQL
SELECT BillingCountry, SUM(Total) AS Gesamtsumme
FROM invoices
GROUP BY BillingCountry
HAVING SUM(Total) > 45;
```

22. Alle Künstler und die Anzahl ihrer Alben, sortiert nach Anzahl der Alben, wenn die Künstler mehr als 3 Alben 
veröffentlicht haben
```SQL
SELECT artists.Name, count(albums.Title) FROM artists
INNER JOIN albums ON albums.ArtistId = artists.ArtistId
GROUP by artists.name
HAVING COUNT(albums.Title) > 3
ORDER BY COUNT(albums.Title) DESC;
```
